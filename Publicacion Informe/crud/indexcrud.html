CRUD en Express.js

El CRUD es un conjunto de operaciones b치sicas utilizadas en las aplicaciones web para gestionar los datos almacenados en una base de datos o en cualquier otro tipo de almacenamiento. En Express.js, puedes implementar estas operaciones utilizando rutas y controladores dedicados para cada acci칩n. Por ejemplo, puedes definir una ruta POST para crear nuevos registros, una ruta GET para leer datos existentes, una ruta PUT para actualizar registros y una ruta DELETE para eliminar registros. Al organizar tu c칩digo de esta manera, puedes mantener una estructura clara y modular en tu aplicaci칩n, lo que facilita su mantenimiento y escalabilidad.

Ejemplo de Crud en Express Js:

                    const express = require('express');
                    const bodyParser = require('body-parser');
                    
                    const app = express();
                    app.use(bodyParser.json());
                    
                    // Base de datos simulada (arreglo)
                    let productos = [];
                    
                    // Ruta para crear un nuevo producto
                    app.post('/productos', (req, res) => {
                      const nuevoProducto = req.body;
                      productos.push(nuevoProducto);
                      res.status(201).json({ mensaje: 'Producto creado exitosamente' });
                    });
                    
                    // Ruta para obtener todos los productos
                    app.get('/productos', (req, res) => {
                      res.json(productos);
                    });
                    
                    // Ruta para actualizar un producto existente
                    app.put('/productos/:id', (req, res) => {
                      const { id } = req.params;
                      const actualizacionProducto = req.body;
                      productos = productos.map(producto => {
                        if (producto.id === id) {
                          return { ...producto, ...actualizacionProducto };
                        }
                        return producto;
                      });
                      res.json({ mensaje: 'Producto actualizado exitosamente' });
                    });
                    
                    // Ruta para eliminar un producto existente
                    app.delete('/productos/:id', (req, res) => {
                      const { id } = req.params;
                      productos = productos.filter(producto => producto.id !== id);
                      res.json({ mensaje: 'Producto eliminado exitosamente' });
                    });
                    
                    // Iniciando el servidor
                    app.listen(3000, () => {
                      console.log('Servidor iniciado en http://localhost:3000');
                    });
