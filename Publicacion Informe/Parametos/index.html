Pase de Parámetros en Express.js

En Express.js, los parámetros pueden pasarse de varias maneras, incluidos los parámetros de ruta, los parámetros de consulta y los datos del cuerpo de la solicitud. Los parámetros de ruta se utilizan para capturar valores específicos de la URL, como identificadores únicos de recursos. Los parámetros de consulta se utilizan para enviar datos adicionales junto con la solicitud, generalmente en forma de pares clave-valor. Los datos del cuerpo de la solicitud se utilizan para enviar información más compleja, como formularios HTML o datos JSON. Express.js proporciona métodos y propiedades en el objeto req (solicitud) para acceder a estos parámetros de manera fácil y segura.

Ejemplo en Express.js:

                    const express = require('express');
                    const app = express();
                    
                    // Ruta con parámetros de ruta
                    app.get('/usuario/:id', (req, res) => {
                      // Capturando el parámetro de ruta ":id" utilizando req.params
                      const userId = req.params.id;
                      // Enviando una respuesta que incluye el ID del usuario capturado
                      res.send(`ID de usuario: ${userId}`);
                    });
                    
                    // Ruta con parámetros de consulta
                    app.get('/busqueda', (req, res) => {
                      // Capturando el parámetro de consulta "q" utilizando req.query
                      const searchTerm = req.query.q;
                      // Enviando una respuesta que incluye el término de búsqueda capturado
                      res.send(`Término de búsqueda: ${searchTerm}`);
                    });
                    
                    // Ruta con datos en el cuerpo de la solicitud (requiere middleware body-parser)
                    app.post('/formulario', (req, res) => {
                      // Capturando los datos del cuerpo de la solicitud utilizando req.body
                      const formData = req.body;
                      // Enviando una respuesta que incluye los datos del formulario capturados
                      res.send(formData);
                    });
                    
                    // Iniciando el servidor en el puerto 3000
                    app.listen(3000, () => {
                      console.log('Servidor iniciado en http://localhost:3000');
                    });
